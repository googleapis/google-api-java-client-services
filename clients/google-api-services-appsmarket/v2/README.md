# Google Workspace Marketplace API Client Library for Java

Lets your Google Workspace Marketplace applications integrate with Google's installtion and licensing services. 

This page contains information about getting started with the Google Workspace Marketplace API
using the Google API Client Library for Java. In addition, you may be interested
in the following documentation:

* Browse the [Javadoc reference for the Google Workspace Marketplace API][javadoc]
* Read the [Developer's Guide for the Google API Client Library for Java][google-api-client].
* Interact with this API in your browser using the [APIs Explorer for the Google Workspace Marketplace API][api-explorer]

## Installation

### Maven

Add the following lines to your `pom.xml` file:

```xml
<project>
  <dependencies>
    <dependency>
      <groupId>com.google.apis</groupId>
      <artifactId>google-api-services-appsmarket</artifactId>
      <version>v2-rev20251217-2.0.0</version>
    </dependency>
  </dependencies>
</project>
```

### Gradle

```gradle
repositories {
  mavenCentral()
}
dependencies {
  implementation 'com.google.apis:google-api-services-appsmarket:v2-rev20251217-2.0.0'
}
```
