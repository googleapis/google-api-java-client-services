# Web Content Publisher API Client Library for Java

webcontentpublisher.googleapis.com API, a service for web content publishers.

This page contains information about getting started with the Web Content Publisher API
using the Google API Client Library for Java. In addition, you may be interested
in the following documentation:

* Browse the [Javadoc reference for the Web Content Publisher API][javadoc]
* Read the [Developer's Guide for the Google API Client Library for Java][google-api-client].
* Interact with this API in your browser using the [APIs Explorer for the Web Content Publisher API][api-explorer]

## Installation

### Maven

Add the following lines to your `pom.xml` file:

```xml
<project>
  <dependencies>
    <dependency>
      <groupId>com.google.apis</groupId>
      <artifactId>google-api-services-webcontentpublisher</artifactId>
      <version>v1-rev20260629-2.0.0</version>
    </dependency>
  </dependencies>
</project>
```

### Gradle

```gradle
repositories {
  mavenCentral()
}
dependencies {
  implementation 'com.google.apis:google-api-services-webcontentpublisher:v1-rev20260629-2.0.0'
}
```
