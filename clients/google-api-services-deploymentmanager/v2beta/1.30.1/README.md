# Google Cloud Deployment Manager API V2Beta Methods Client Library for Java

The Deployment Manager API allows users to declaratively configure, deploy and run complex solutions on the Google Cloud Platform.

This page contains information about getting started with the Google Cloud Deployment Manager API V2Beta Methods
using the Google API Client Library for Java. In addition, you may be interested
in the following documentation:

* Browse the [Javadoc reference for the Google Cloud Deployment Manager API V2Beta Methods][javadoc]
* Read the [Developer's Guide for the Google API Client Library for Java][google-api-client].
* Interact with this API in your browser using the [APIs Explorer for the Google Cloud Deployment Manager API V2Beta Methods][api-explorer]

## Installation

### Maven

Add the following lines to your `pom.xml` file:

```xml
<project>
  <dependencies>
    <dependency>
      <groupId>com.google.apis</groupId>
      <artifactId>google-api-services-deploymentmanager</artifactId>
      <version>v2beta-rev20181207-1.30.3</version>
    </dependency>
  </dependencies>
</project>
```

### Gradle

```gradle
repositories {
  mavenCentral()
}
dependencies {
  compile 'com.google.apis:google-api-services-deploymentmanager:v2beta-rev20181207-1.30.3'
}
```
